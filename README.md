# ai-dev-toolkit

AI development assets for the Fundzflow project — clone this repo to get all three components.

## Contents

| Folder | Purpose |
|---|---|
| `ai-toolkit/` | Python toolkit with specs, tests, prompts, and automation scripts |
| `ai-framework/` | Coding standards, templates, checklists, and workflow guides |
| `ai-ProjectSpecs/` | Project context, architecture notes, sprint plans, and decision logs |

## Usage

```bash
git clone https://github.com/kaaldev73/ai-dev-toolkit.git
```
