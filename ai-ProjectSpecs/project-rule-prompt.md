# Project Rule Prompt

Use this file to write the prompt you give to an AI model to enforce project rules and business constraints.

Paste the contents of `project-rules.md` into your AI session, or reference this prompt to instruct the AI to read it.

---

## Prompt Template

```
Before making any changes, read and follow these project rules:

[paste contents of project-rules.md here]

These rules are mandatory. Do not invent or modify business rules.
Do not begin implementation until you confirm you understand and will follow these rules.
```
