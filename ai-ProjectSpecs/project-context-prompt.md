# Project Context Prompt

Use this file to write the prompt you give to an AI model at the start of a session to load project context.

Paste the contents of `project-context.md` into your AI session, or reference this prompt to instruct the AI to read it.

---

## Prompt Template

```
You are working on [YOUR PROJECT NAME], a [brief description].

Read the following project context before proceeding:

[paste contents of project-context.md here]

Do not begin any implementation until you confirm you understand the project context.
```
